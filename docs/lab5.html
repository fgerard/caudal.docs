Lab 5 - Caudal Integration

A lab exercises intended to show how to integrate Caudal with Elasticsearch database and another Caudal nodes

Requirements

  • Have successfully completed Lab 4: Correlation Streamers

Setting up an event enricher streamer

  1. Change current directory to the caudal-labs project
  1. Edit config/caudal-config.clj file to configure a dafault streamer that transform incomming messages and send them to the Elasticsearch database.

Starting Elasticsearch

  1. Open a new terminal an change current directory to the one containing elasticsearch installation.
  1. Start Elasticsearch database
  1. Clean Elasticsearch labs index

Feeding event streamer

  1. Restart Caudal for applying changes in configuration.

  1. Open another terminal and send through the tcp channel the events shown below.

  2. Verify the generated log for the received events

Verifying database contents

  1. Open a new terminal and execute the query command shown for checking the events previously processed.

Starting Kibana visualizer

  1. Open a new terminal an change current directory to the one containing kibana installation.
  1. Start Elasticsearch database

  1. Open url http://localhost:5601 in a browser and click on Settings.

  2. Set the value labs* in the field Index name or pattern and select the value time in Time-field name field.

  1. Click on button Create.

Creating Pie visualization for event data generated

  1. Verify that visualizations date range is correct.

  2. Click on Visualize option and then in Pie chart.

  1. Select From new search option.

  1. Click on Split Slices buckets type.

  1. Select Range in Aggregation section and ammount in Field. Add the four ranges shown in image below then Apply changes.

  1. Once you entered ranges, must save the visualization. For that click on Save Visualization, set By ammount value into Title field, then click button Save.

Creating Vertical bar chart visualization for event data generated

  1. Click on Visualize option and then in Vertical bar chart.

  1. Select From new search option.

  1. Expand Y-Axis in metrics section and fill the fields as shown.

  1. Click X-Axis in buckets section, fill the fields as shown.

  1. Press the Apply changes button.

  1. Once you entered ranges, must save the visualization. For that click on Save Visualization, set Totals by product value into Title field, then click button Save.

Creating a dashboard

  1. Click on Dashboard option.

  1. Click on + button to add a new dashboard.

  1. Select visualizations previously created and resize them as shown.

  1. Click on Save dashboard button.

  1. Set Sample Dashboard as the name for the one just created.

  1. Created dashboard should appear as shown below.
